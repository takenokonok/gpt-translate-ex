プライバシーポリシー

最終更新日: 2026-05-16

本ページは Chrome 拡張機能「タブ音声リアルタイム吹き替え」（以下「本拡張機能」）に関するプライバシー情報です。

開発者

  • 開発者: kotaro.otake
  • 連絡先: kotaro.otake@takenokonoko.com

収集するデータ

本拡張機能は、開発者のサーバーにユーザー情報・音声・字幕などを送信する機能を一切持ちません。 すべての処理はユーザー自身のブラウザ内で行われ、OpenAI API への通信もユーザー環境から直接行われます。

項目 取り扱い
OpenAI API キー ユーザーが拡張のポップアップに入力したキーは、ユーザー本人のブラウザ内（chrome.storage.local）にのみ保存され、音声・字幕の処理のために OpenAI の API（api.openai.com）へユーザー環境から直接送信されます。開発者がキーを収集・保管・閲覧することはありません。
タブの音声 ユーザーが「開始」を押したセッションの間に限り、Chrome の tabCapture 機能でキャプチャされた音声がブラウザ内で処理され、OpenAI の Realtime Translation API へ WebRTC 経由で送信されます。セッション終了後は破棄されます。
設定情報 翻訳先言語、元音声・吹替の音量、字幕表示の ON/OFF、字幕パネルの位置とサイズ。これらはユーザーのブラウザ内（chrome.storage.local）にのみ保存されます。
閲覧履歴・個人を特定できる情報の販売・第三者提供 行いません。

権限の利用目的

権限 用途
storage API キー、言語、音量、字幕表示などの設定をローカルに保存するため。
tabCapture 現在のタブから音声を取得し、リアルタイム翻訳の入力として使うため。
activeTab ユーザーが操作中のタブとの連携（音量制御・字幕表示）のため。
offscreen Service Worker では動かせない WebRTC / MediaStream を Offscreen Document で実行するため。
scripting 字幕オーバーレイ表示と元音声の音量制御のためのコンテンツスクリプトを動的に注入するため。
host_permissions: api.openai.com/* OpenAI API への直接通信のため。
host_permissions: *://*/* 任意のウェブページ上で字幕オーバーレイを表示し、元音声の音量を制御するため（ユーザー操作で開始した時のみ動作）。

第三者サービス（OpenAI）

本拡張機能は OpenAI の Realtime Translation API を利用します。音声データおよび関連メタデータの取り扱いは、 OpenAI のプライバシーポリシー に従います。ユーザーは OpenAI の利用規約およびポリシーを確認のうえ API キーを利用してください。

データの保持と削除

  • ローカルに保存される設定は、ユーザーが拡張機能を削除する、または chrome://extensions から「拡張機能のデータをすべて削除」を実行することでいつでも消去できます。
  • セッション中にやり取りされる音声・字幕データはブラウザのメモリ上に存在し、停止時または拡張のリロード時に破棄されます。

変更履歴

ポリシーに重要な変更があった場合は、本ページ上部の「最終更新日」を改訂します。
